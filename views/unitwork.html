This next lab here was about a very common loop in C which is a for loop, you will almost always find for loops in C code as they are efficent and precise in sorting and printing information.

This gist is about another common algorithm in C which is a binary search. Binary searches sort through an array of sorted numbers by finding the middle of the array, and if the number is smaller than the middle number, it will go to the middle of the first half of the array, and that will continue until the number was found. This was a challenge to code since I had to get all the algorithms right and made sure they did the right thing.
